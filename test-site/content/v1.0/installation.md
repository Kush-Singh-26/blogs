# Installation v1.0

Install Kosh v1.0 on your system.

> **Warning:** This page only exists in v1.0. For the latest installation, see [Installation](../installation.md).

## Requirements

- Go 1.18 or later
- Git

## Install

```bash
go install github.com/kosh/kosh@v1.0
```

## Verify

```bash
kosh version
# Output: Kosh v1.0.0
```

## Platform Notes

### macOS / Linux

```bash
go install github.com/kosh/kosh@v1.0
```

### Windows

```powershell
go install github.com/kosh/kosh@v1.0
```

## v2.0 Highlights

### Version Support

v2.0 introduced documentation versioning:

- Multiple documentation versions
- Version-specific navigation
- Outdated version banners

### Search Engine

WASM-powered client-side search:

- BM25 ranking algorithm
- Snippet extraction
- Keyboard navigation

### Theme Features

- Recursive sidebar navigation
- Version selector
- Breadcrumb navigation
